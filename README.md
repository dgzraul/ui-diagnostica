## Proceso de incio
Se deberá instalar las dependencias necesarias, ya que el proyecto requiere de algunas librerias. 

``
  $ npm install
``

## Ejecución
El script para iniciar el proceso de desarrollo.
```
  $ npm run start:dev
```
